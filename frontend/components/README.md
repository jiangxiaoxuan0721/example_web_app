# Components 组件库

可复用的前端组件集合。

## 组件列表

| 组件 | 功能 | 文件 |
|-------|------|------|
| kpi_cards | KPI 指标展示 | kpi_cards.css, kpi_cards.js |
| tables | 数据表格（搜索/排序/分页） | tables.css, tables.js |
| drawer | 侧边详情面板 | drawer.css, drawer.js |
| picture | 图片/HTML 内容展示 | picture.css, picture.js |
| buttons | 按钮组件 | buttons.css, buttons.js |
| function_cards | 功能卡片 | function_cards.css, function_cards.js |
| informations | 信息展示组件 | informations.css, informations.js |

## 组件规范

每个组件必须包含：
- 一个 CSS 文件：组件样式
- 一个 JS 文件：组件逻辑（类定义）

## 使用方法

在页面 HTML 中引入：
```html
<link rel="stylesheet" href="/components/component_name/component_name.css">
<script src="/components/component_name/component_name.js"></script>
```
